🐢 Turtle Crossing Game
A simple arcade game built with Python's turtle module. You're a turtle trying to cross a busy road — dodge the cars and make it to the other side!
How to Play

Press Up arrow to move forward
Dodge the cars coming from the right
Each time you make it across, the cars get faster
If a car hits you, game over

How to Run
Make sure you have Python installed, then just run:
python main.py
No extra libraries needed, everything uses Python's built-in turtle module.
Project Structure
├── main.py          # runs the game loop
├── player.py        # player movement and position
├── car_manager.py   # car spawning and speed
├── scoreboard.py    # level display and game over screen
